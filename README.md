# Islam Glab | Personal Link Hub

بوابة شخصية للروابط والأدوات الرقمية الأساسية مع شعارات رسمية لكل خدمة

## ✨ المميزات الجديدة

### 🎨 الشعارات الرسمية المحدثة

- **شعارات رسمية حقيقية**: كل رابط يعرض الشعار الرسمي الحقيقي للشركة/الخدمة
- **جودة عالية محسنة**: شعارات SVG عالية الجودة محدثة حديثاً
- **ألوان أصلية دقيقة**: الألوان الرسمية الدقيقة لكل علامة تجارية
- **تجربة احترافية متطورة**: مظهر احترافي ومتسق مع أحدث المعايير

### 🔍 البحث والتصفية

- **بحث ذكي**: ابحث في جميع الروابط بسهولة
- **تصفية بالفئات**: تصفية الروابط حسب النوع
- **نتائج فورية**: نتائج البحث تظهر فوراً

### 📱 تصميم متجاوب

- **جميع الأجهزة**: يعمل على الهواتف والأجهزة اللوحية والحواسيب
- **تجربة سلسة**: تصميم مخصص لكل حجم شاشة
- **سهولة الاستخدام**: واجهة بديهية وسهلة الاستخدام

### ⚡ أداء محسن

- **تحميل سريع**: شعارات SVG خفيفة الحجم
- **PWA**: تطبيق ويب تقدمي يمكن تثبيته
- **عمل بدون إنترنت**: يعمل حتى بدون اتصال بالإنترنت

## 🗂️ الفئات المتاحة

### 🌟 مصنع النجمة

- **بحث جوجل دوكس**: أداة البحث في مستندات جوجل
- **تنسيق الأرقام**: أداة تنسيق وترتيب الأرقام
- **مصنع النجمة الرئيسي**: الصفحة الرئيسية لمصنع النجمة
- **مصنع النجمة - صفحة 2**: الصفحة الثانية لمصنع النجمة
- **الراوتر**: إعدادات الراوتر والشبكة
- **الأرشيف**: أرشيف مصنع النجمة
- **مصنع النجمة - صفحة 3**: الصفحة الثالثة لمصنع النجمة
- **جوجل درايف**: تخزين ملفات مصنع النجمة

## 🚀 كيفية التشغيل

### المتطلبات

- Node.js (الإصدار 18 أو أحدث)
- متصفح ويب حديث

### التثبيت والتشغيل

```bash
# تثبيت التبعيات
npm install

# تشغيل التطبيق محلياً
npm start

# أو تشغيل الخادم مباشرة
node server.js
```

### الوصول للتطبيق

افتح المتصفح وانتقل إلى: `http://localhost:3000`

## 🛠️ التقنيات المستخدمة

### Frontend

- **HTML5**: هيكل الصفحة
- **CSS3**: التصميم والتنسيق
- **JavaScript (ES6+)**: التفاعل والوظائف
- **Font Awesome**: الأيقونات

### Backend

- **Node.js**: بيئة التشغيل
- **Express.js**: إطار العمل للخادم

### الأدوات الإضافية

- **PWA**: تطبيق ويب تقدمي
- **Service Worker**: التخزين المؤقت والعمل بدون إنترنت
- **Web Manifest**: بيانات التطبيق
- **SEO**: تحسين محركات البحث
